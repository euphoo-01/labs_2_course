# Лабораторные работы 2 курс
### Дисциплины:
- Конструирование программного обеспечения (C++)
- Скриптовые языки программирования (Legacy JavaScript)
- Алгоритмы и структуры данных (C++)
- Объектно-ориентированное программирование (C#)
- Компьютерные системы и сети (C++, Cisco Packet Tracer)

### Курсовой проект:
#### Краткие сведения:
Компилятор для собственного языка программирования. 
Язык разработки: **C++**. 
Выходной код: **NASM**.
Основан на конечных автоматах, автоматах с магазинной памятью, дереве разбора.
Использует обратную польскую нотацию для вычисления выражений.
